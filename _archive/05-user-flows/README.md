## Tổng quan

Tài liệu này chứa các user flows hoàn chỉnh - mỗi file mô tả một flow từ đầu đến cuối.

## Cấu trúc

### Student Flows
- [Student Learning Flow](./student-learning-flow.md) - Flow học tập hoàn chỉnh (từ learning plan đến practice)
- [Mini Test Flow](./mini-test-flow.md) - Flow mini test (unlock → làm bài → kết quả)

### Parent Flows
- [Parent View Flow](./parent-view-flow.md) - Flow xem báo cáo, linking

## Quy tắc

### Mỗi file = 1 flow hoàn chỉnh
- Không trộn nhiều flow trong 1 file
- Flow phải có đầu và cuối rõ ràng
- Mô tả các bước chi tiết

### Mapping với Domain Model
- Flows phải map về [Domain Model](../02-domain-model/)
- Sử dụng đúng terminology (Chapter, Skill, etc.)

### Mapping với Product Rules
- Flows phải tuân theo [Product Rules](../03-product-rules/)
- Business rules được áp dụng trong flows
