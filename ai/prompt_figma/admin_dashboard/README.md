## NGUYÊN TẮC THIẾT KẾ

### Admin Dashboard (Web)
- **Màu sắc:** Chuyên nghiệp, tin cậy, data-focused
- **Typography:** Dễ đọc, hierarchy rõ ràng, font size phù hợp
- **Data Visualization:** Biểu đồ rõ ràng, dễ hiểu, có legend
- **Layout:** Responsive, tối ưu cho desktop (1440px+)
- **Language:** Tiếng Việt, thuật ngữ kỹ thuật được giải thích
- **Interaction:** Hover effects, click feedback, loading states
